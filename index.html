One-time setup: connect your Supabase project

This links the portal to your free cloud database so every participant's sheets are stored centrally.

Paste the two values from your Supabase project (Settings → API Keys):

How do I get these + set up the database?
  1. Go to supabase.com → create a free account → New project.
  2. Open SQL Editor, paste the SQL from the setup guide, and run it (creates the sheets table + security rules + dashboard support).
  3. Project URL: look at your browser's address bar on any project page — it reads .../project/YOUR-ID/.... Your URL is https://YOUR-ID.supabase.co.
  4. Key: open Settings → API Keys and copy the Publishable key (starts sb_publishable_). The older anon key (starts eyJ, under the "Legacy" tab) also works.
  5. For password security, in Authentication → Providers → Email turn off "Confirm email" for quick internal use, or leave it on to require email confirmation.

Once connected, the values are remembered in this browser. To bake them in permanently for everyone, edit the SUPABASE_URL / SUPABASE_KEY lines near the top of the script in this file before deploying.
